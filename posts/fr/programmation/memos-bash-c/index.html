Commandes Terminal

Commande Nom de la commande Effet Exemple
man Manuel Ouvre une page du manuel à la page donnée man man
ls Listing Liste les fichiers du répertoire donné (ou courant si aucun répertoire donné) ls /
cd Change Directory Modifie le répertoire courant au chemin indiqué cd /tmp
pwd Print Working Directory Affiche le répertoire courand pwd
echo Echo Écrit dans la sortie standard une chaîne de charactères echo "Hello, World!"
cat Concatenate Affiche le contenu d’un fichier cat hello.txt
grep Grep Recherche schéma dans un fichier ou un flux d’entrée `lspci
nano GNU nano Un éditeur de texte en terminal nano file.txt

Flux de redirection

Symbole Effet Exemple
> Écrit la sortie standard dans un fichier echo "Hello, World!" > hello.txt
>> Ajoute la sortie standard à la fin du fichier echo "Hello, World!" >> hello.txt
< Écrit le contenu d’un fichier dans l’entrée standard d’un programme grep -i ".txt" < liste.txt
` ` (Pipe) Lie la sortie standard d’un programme à l’entrée d’un autre

Langage C

Syntaxe

Variable

type1 v1;       // Déclaration de la variable v1 avec le type type1
v1 = 0;         // Assignation de la variable v1
type2 v2 = 0;   // Déclaration et assignation de la variable
                // v2 de type type2 à la valeur 0

Fonction

r_type func_name(p1_type p1, p2_type p2, ...) {
    /* 
    *  Corps de la fonction
    *  ...
    */
    return r_val;
}
r_type
type de retour de la fonction. Cela peut être une primitive ou une structure.
func name
nom de la fonction. Doit commencer par une lettre et peut contenir _ et des chiffres.
r_val
valeur de retour de la fonction. Peut être une variable, ou une expression.
